File Format Guide

Required Columns

  • Date — YYYY-MM-DD format
  • Item Name — e.g. Rice, Vegetables
  • Quantity — numeric value
  • Price — price per unit in Rs
  • Nationality — Bangladeshi / Sri Lankan / Indian / Malagasy

Unit Detection

  • Oil → measured in Litres (L)
  • Bread → measured in pieces (pc)
  • Eggs → measured in units
  • Other items → measured in kg
